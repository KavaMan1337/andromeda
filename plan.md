## Фаза 2 — Анимации появления (улучшенные)

### 2.1 Улучшить hero
- Добавить "дышащий" glow-эффект на логотип (более плавный, чем current)
- Subtle зеркальное свечение за логотипом
- Заголовок появляется с небольшим scale (0.95 → 1)
- Subtitle появляется с fade-up после заголовка

### 2.2 Scroll-triggered fade-in для секций
Добавить IntersectionObserver (минимальный JS) для:
- Секции появляются только когда входят в viewport
- Fade + translateY при скролле
- Не переигрывать анимацию (once: true)

### 2.3 Улучшить stagger на карточках
Текущий: `opacity:0` + stagger 0.7s. Добавить:
- Subtle scale: 0.95 → 1 вместе с opacity
- Stagger delay увеличить до 0.15s (медленнее, эффектнее)
- `.stagger-item` получает плавный ease-out
